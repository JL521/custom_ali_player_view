# custom_ali_player_view

基于阿里播放器进行的二次封装和优化、完善优化阿里播放器

1、支持界面自定义宽高、随处可以使用
2、支持视频全屏播放、内部实现全屏逻辑
3、支持播放器多页面实例化
4、统一管理播放器，保证播放器ID的唯一性
